# Parameter Evidence

Track 11 does not add standalone YAML parameter files.

The modeled compliance timeline uses person-level input variable
`tax_admin__automatic_tax_assessment`. Manual filing deadlines are represented
by `tax_admin__filing_deadline_months_after_year_end`.

Current modeled values:

- Automatic assessment: `0` manual filing months.
- Manual filing: `3` months after year end.
